A clone of the popular game Wordle


This is a carbon copy of the popular game Wordle made using Java, what makes this clone different from the original is the fact that the user isn't limited to one word a day! The user will be able to get a new word when they run out of tries or when they guess the word correctly!


Check out the source code on Github

A simple Pomodoro Timer Mobile App


This is a simple mobile app that works as a Pomodoro Timer this app was made using React Native. It takes users time input for a time for work and a time for break, then will start a work timer, upon completion the uwser is notified of the timers completion, and then the break timer starts when the user acknowledges that the work timer is completed.
